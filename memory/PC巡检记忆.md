# PC巡检记忆

## 任务描述

- **名称**：PC巡检
- **调度**：每30分钟运行一次
- **内容**：
  - `df -h`：检查磁盘空间
  - `free -h`：检查内存使用情况
  - `ollama list`：列出OLLAMA模型
  - `nvidia-smi`：检查GPU状态
  - `ls -lhS /tmp | head -5`：列出/tmp目录下最大的文件
- **只列异常项**

## 路由器巡检记忆

- **名称**：路由器巡检
- **调度**：每60分钟运行一次
- **内容**：
  - SSH到mt3600be(192.168.8.1)
  - 检查uptime、内存、SSD挂载、AGH状态、swap、logread异常
- **只列异常项**
