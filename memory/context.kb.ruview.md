## 核心

WiFi 信号 → CSI 信道状态信息提取 → 实时空间智能：
- 存在检测（房间里有没有人）
- 生命体征（呼吸频率、心率）
- 位置追踪
- 不需要摄像头，不需要额外硬件

## 原理

人体呼吸/移动微扰 WiFi 信号 → CSI 分析 → AI 模型识别模式。MIT 2015 WiFi 穿墙奠基。

## 对雅典娜

- MT3600BE(MT7988) 和 RAX3000Me(MT7981) 理论上能采 CSI
- 需要特定内核模块 + 驱动支持
- 门槛高，暂不启动
