# 脚本工具清单

## 位置索引
| 路径 | 用途 |
|------|------|
| `~/scripts/switch-model.sh` | 切换 OpenClaw 模型 (flash/pro) |
| `C:\Users\lz\Desktop\xmir-patcher-main\` | 小米路由器漏洞利用工具 |
| `~/.ssh/router_new` | MT3600BE SSH 私钥 |
| `/mnt/g/backups/openclaw/` | G: 归归档 主备份目录 |
| `/mnt/sda1/backups/` | 路由器 SSD 冗余备份 |

## switch-model.sh
```bash
~/scripts/switch-model.sh flash   # 切到 deepseek-v4-flash
~/scripts/switch-model.sh pro     # 切到 deepseek-v4-pro
```

## xmir-patcher (Windows)
- 位置: `C:\Users\lz\Desktop\xmir-patcher-main\`
- 入口: `!START.bat`
- 功能: SSH 漏洞利用, 改密码, 备份固件
- 菜单 8→2: 改 root 密码
- 密码算法: `gateway.py` line 262-272 (MD5(SN + reversed_guid)[:8])

## 路由器 SSH
```bash
# MT3600BE (密钥登录)
ssh -i ~/.ssh/router_new root@192.168.8.1

# RA80 小米 (密码登录)
ssh -o HostKeyAlgorithms=+ssh-rsa root@192.168.8.178
# 密码: Aa123456
```

## 服务管理
```bash
# Gateway
systemctl --user restart openclaw-gateway
systemctl --user status openclaw-gateway
tail -f /tmp/openclaw/openclaw-*.log

# Ollama
ollama serve > /tmp/ollama.log 2>&1 &
kill $(pgrep ollama)
```

## 备份命令
```bash
# 手动备份到 G: 归归档 (主)
tar -czf /mnt/g/backups/openclaw/backup_$(date +%Y%m%d).tar.gz -C ~/.openclaw/workspace .

# 手动备份到路由器 SSD (冗余)
tar -czf /tmp/backup.tar.gz -C ~/.openclaw/workspace MEMORY.md memory/
scp -i ~/.ssh/router_new /tmp/backup.tar.gz root@192.168.8.1:/mnt/sda1/backups/
```
