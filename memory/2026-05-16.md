### OpenClaw 文档自学 (11:24 - 11:31)
- 系统阅读了 OpenClaw v2026.5.12 本地文档
- 已学习的模块和关键发现：

| 模块 | 关键发现 |
|:---|:---|
| **Agent Loop** | 7 步：intake → context assembly → model inference → tool execution → streaming → persistence。agent 和 agent.wait 两种入口 |
| **Gateway Config** | JSON5 格式，`openclaw.json`，支持热重载，严格 schema 校验 |
| **QQ Bot** | 插件安装 → QQ 开放平台注册 → AppID/AppSecret → channels add。支持 C2C/群聊/频道/富媒体 |
| **Tools** | 10 个内置类别：Runtime/Files/Web/Browser/Messaging/Sessions/Automation/Gateway/Media/Tool Search |
| **Exec** | 支持 host(sandbox/gateway/node)、pty 模式、background/yieldMs、timeout、env 覆盖 |
| **Sub-agents** | `sessions_spawn` + `sessions_yield` 正确用法。隔离运行，完成自动通知。支持 fork/nesting |
| **Skills** | 6 级优先级（workspace > .agents > ~/.agents > managed > bundled > extraDirs）。Agent allowlist 独立控制 |
| **Browser** | 专用 openclaw profile，橙色标签，与用户 Edge 隔离。支持多 profile。内置 browser-automation skill |
| **Multi-agent** | 同一 Gateway 进程内逻辑隔离。独立 workspace + agentDir + 会话。binding 映射渠道到 agent |
| **Memory** | MEMORY.md 启动加载，memory/YYYY-MM-DD.md 按需搜索。memory_search 用混合搜索（向量+关键词） |
| **Heartbeat** | 默认 30m 巡检，HEARTBEAT_OK 静默。target 可配 none/last。isolatedSession 不送历史 |
| **Cron** | 4 种执行：main/isolated/current/session:xxx。支持 at/every/cron 表达式。one-shot 自动删 |
| **Sandbox** | 3 种模式（off/non-main/all）+ 3 种后端（docker/ssh/openshell）。session/agent/shared 三种作用域 |
| **Context** | `/context list|detail|map` 看占用。tool schema JSON 也算 token。超出窗口自动截断 |
| **Failover** | 两层：Auth profile 轮换 → Model 回退链。手动选模型不回退，默认值自动 fallback |
