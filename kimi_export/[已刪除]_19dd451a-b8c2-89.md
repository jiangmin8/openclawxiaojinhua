让我提供查看显存的方法。查看**显存**（GPU 显存）：

---

**方式1：任务管理器（最简单）**
```
Ctrl + Shift + Esc → 性能 → GPU
```
看"专用 GPU 内存"那一栏。

---

**方式2：命令行**

```powershell
# NVIDIA 显卡
nvidia-smi

# 或详细点
nvidia-smi -q -d MEMORY
```

---

**方式3：PowerShell**

```powershell
# 看显卡信息
Get-WmiObject Win32_VideoController | Select Name, AdapterRAM

# 或
Get-CimInstance Win32_VideoController | Select Name, @{N="显存(GB)";E={[math]::Round($_.AdapterRAM/1GB,2)}}
```
