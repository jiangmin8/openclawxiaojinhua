## 完整操作流程


```bash

```

如果 `config.yaml` 已经存在，先备份：

```bash
cp /home/lz/.hermes/config.yaml /home/lz/.hermes/config.yaml.bak.$(date +%Y%m%d%H%M%S)
```

---

### 2. 追加配置

```bash
cat >> /home/lz/.hermes/config.yaml << 'EOF'

persona:
  soul_path: /home/lz/.hermes/persona/SOUL.md
  auto_load: true

memory:
  backend: chroma
  chroma:
    path: /home/lz/.hermes/memory/chroma
    collection: hermes_longterm

tools:
  search:
    default: duckduckgo
  reader:
    default: jina
  voice:
    stt: whisper
    tts: edge-tts
    tts_voice: zh-CN-XiaoxiaoNeural
EOF
```

```bash
    "https://mirror.baidubce.com"
```

## 如果还是不行（镜像站也失效）

用你主路由的代理，给 Docker 配 HTTP 代理：

```bash
# 先确认 Windows 宿主机 IP
ip route | grep default | awk '{print $3}'
```

假设是 `172.24.176.1`，代理端口 `7890`：

```bash
sudo mkdir -p /etc/systemd/system/docker.service.d
sudo tee /etc/systemd/system/docker.service.d/proxy.conf << EOF
[Service]
Environment="HTTP_PROXY=http://172.24.176.1:7890"
Environment="HTTPS_PROXY=http://172.24.176.1:7890"
Environment="NO_PROXY=localhost,127.0.0.1"
EOF

sudo systemctl daemon-reload
sudo systemctl restart docker
docker run hello-world
```
