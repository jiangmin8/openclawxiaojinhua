## Hermes Agent 核心技能体系

### 1. **自动技能创建（Skill Creation）**
这是 Hermes 的核心特性。当完成复杂任务（通常涉及 5+ 个工具调用）后，Agent 会自动将经验抽象为**可复用的技能文档** [^4^]。技能存储在 `~/.hermes/skills/` 目录下，采用渐进式披露模式减少 Token 消耗：

| 层级 | 内容 | Token 占用 |
|------|------|-----------|
| Level 0 | 技能名称和描述列表 | ~3,000 tokens |
| Level 1 | 特定技能的完整内容 | 按需加载 |
| Level 2 | 技能内的具体参考文件 | 精确加载 |

### 2. **持久记忆系统**
- **MEMORY.md**（2,200 字符）：存储环境事实、约定和经验教训
- **USER.md**（1,375 字符）：记录用户偏好和沟通风格
- **SQLite FTS5 全文搜索**：可搜索所有历史会话 [^2^]

### 3. **内置工具集（Toolsets）**
Hermes 内置了丰富的工具集合，可按平台启用/禁用 [^4^]：

| 工具类别 | 功能 |
|---------|------|
| **Web Search** | 网络搜索 |
| **Terminal** | 终端命令执行 |
| **File Edit** | 文件编辑 |
| **Memory** | 记忆管理 |
| **Delegation** | 子代理委派 |
| **Browser** | 浏览器自动化（支持 Browserbase、本地 Chrome CDP、Chromium）|
| **Vision** | 图像粘贴和分析 |
| **Image Gen** | 图像生成（FAL.ai FLUX 2 Pro）|
