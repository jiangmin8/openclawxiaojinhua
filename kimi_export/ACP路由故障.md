nameserver 8.8.8.8

## 更直接：检查 Windows 防火墙对 WSL 的影响

WSL2 使用虚拟网卡，Windows 防火墙可能拦截。

在 **Windows PowerShell（管理员）** 里执行：

```powershell
# 查看 WSL 相关防火墙规则
Get-NetFirewallRule | Where-Object { $_.DisplayName -like "*WSL*" -or $_.DisplayName -like "*Hyper-V*" } | Select-Object DisplayName, Enabled, Direction, Action

# 查看当前活动的防火墙规则
Get-NetFirewallRule -Enabled True | Where-Object { $_.Direction -eq "Outbound" } | Select-Object DisplayName, Action | Sort-Object DisplayName
```
