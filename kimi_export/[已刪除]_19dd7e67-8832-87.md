## 推荐方案：每天一次自动备份 + 手动快捷备份

### 1. 备份脚本（保存到 `~/backup-openclaw.sh`）

```bash
cat > ~/backup-openclaw.sh << 'EOF'
#!/bin/bash
# OpenClaw 备份脚本

SOURCE=~/openclaw
DEST=/mnt/f/github仓库/openclaw-backup
DATE=$(date +%Y%m%d_%H%M%S)
LOG=~/openclaw-backup.log

# 创建带时间戳的版本备份
mkdir -p "$DEST"

# rsync 同步（排除 node_modules、.git、dist 等构建产物）
rsync -av --delete \
  --exclude='node_modules' \
  --exclude='.git' \
  --exclude='dist' \
  --exclude='dist-runtime' \
  --exclude='*.log' \
  --exclude='.pnpm-store' \
  "$SOURCE/" "$DEST/"

# 可选：创建压缩归档
# tar czf "$DEST/backup_$DATE.tar.gz" -C "$SOURCE" .

echo "[$(date)] Backup completed: $SOURCE → $DEST" >> "$LOG"
EOF

chmod +x ~/backup-openclaw.sh
```

### 2. 手动执行（随时备份）
