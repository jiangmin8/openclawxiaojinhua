## 方案一：使用"悟空"一键脚本（推荐新手）

这是最简单的方式，由 B 站 UP 主"悟空的日常"开发的专用脚本。

### 1. SSH 连接到路由器
```bash
ssh root@192.168.8.1
```

### 2. 执行 MT3600BE 专用脚本
```bash
sh -c "$(curl -fsSL https://cafe.cpolar.cn/wkdaily/gl/raw/branch/main/mt3600.sh)"
```

### 3. 选择功能
脚本运行后会出现菜单，选择：
- **选项 1**：一键 iStoreOS 风格化（安装 iStore 商店）
- **选项 3**：单独安装 iStore 商店

### 4. 安装 PassWall
脚本安装完成后，需要手动下载 PassWall 的 ipk 包安装（见方案二步骤）

---

## 方案二：手动安装 PassWall（稳定可靠）

### 步骤 1：下载所需文件

从 GitHub 下载以下文件（MT3600BE 对应 aarch64_cortex-a53 架构）：

| 文件 | 下载地址 |
|------|---------|
| PassWall 主程序 | https://github.com/Openwrt-Passwall/openwrt-passwall/releases |
| 依赖包 | `passwall_packages_ipk_aarch64_cortex-a53.zip` |
| 中文语言包 | `luci-i18n-passwall-zh-cn_xx_all.ipk` |
| 运行时依赖 | `luci-lua-runtime_all.ipk` |
| OpenSSL 库 | `libopenssl3.ipk` |

### 步骤 2：安装依赖
```bash
# 下载悟空提供的依赖包
wget "https://cafe.cpolar.cn/wkdaily/gl/raw/branch/main/theme/luci-lua-runtime_all.ipk" -O /tmp/luci-lua-runtime_all.ipk
wget "https://cafe.cpolar.cn/wkdaily/gl/raw/branch/main/theme/libopenssl3.ipk" -O /tmp/libopenssl3.ipk

# 安装依赖
opkg update
opkg install /tmp/luci-lua-runtime_all.ipk /tmp/libopenssl3.ipk
```

### 步骤 3：安装 PassWall
```bash
# 进入存放 ipk 文件的目录
cd /tmp/upload  # 或你上传文件的目录

# 解压依赖包
unzip passwall_packages_ipk_aarch64_cortex-a53.zip

# 安装所有 ipk 包
opkg install *.ipk --force-reinstall
```

### 步骤 4：重启路由器
```bash
reboot
```

重启后在 **服务 → PassWall** 中配置节点即可使用。

## 方案三：使用 .run 离线安装包（最简单）

第三方开发者编译的整合包，适合无法科学上网的环境。

### 下载地址
https://github.com/wkccd/CloudRunFilesBuilder/releases

**注意**：下载文件名包含 `aarch64_a53` 结尾的版本

### 安装方法
```bash
# 上传 .run 文件到路由器 /tmp 目录
cd /tmp
sh 包名.run
```

## 方案四：安装 OpenClash

### 方法 1：通过 ipk 安装
```bash
# 下载 OpenClash
cd /tmp
wget https://github.com/vernesong/OpenClash/releases/download/v0.46.075/luci-app-openclash_0.46.075_all.ipk -O openclash.ipk

# 安装依赖
opkg update
opkg install bash iptables dnsmasq-full curl ca-bundle ipset ip-full iptables-mod-tproxy iptables-mod-extra ruby ruby-yaml kmod-tun kmod-inet-diag unzip luci-compat luci luci-base

# 安装 OpenClash
opkg install openclash.ipk
```

### 方法 2：通过软件源安装（需要科学上网）
```bash
# 添加 PassWall 软件源
wget https://free.nchc.org.tw/osdn/storage/g/o/op/openwrt-passwall-build/passwall.pub
opkg-key add passwall.pub

# 添加仓库
echo "src/gz passwall_packages https://free.nchc.org.tw/osdn/storage/g/o/op/openwrt-passwall-build/releases/packages-21.02/aarch64_cortex-a53/passwall_packages" >> /etc/opkg/customfeeds.conf
echo "src/gz passwall_luci https://free.nchc.org.tw/osdn/storage/g/o/op/openwrt-passwall-build/releases/packages-21.02/aarch64_cortex-a53/passwall_luci" >> /etc/opkg/customfeeds.conf

# 更新并安装
opkg update
opkg install luci-app-passwall
```
