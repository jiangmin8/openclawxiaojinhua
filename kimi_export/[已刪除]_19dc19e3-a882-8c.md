创建 Windows 11 本地账户，避开微软账户强制登录，有以下几种方法：

---

## 方法一：安装时直接创建（最干净）

**在联网步骤断网跳过：**

1. 安装到"让我们为你连接到网络"界面时，按 `Shift + F10` 打开 CMD
2. 输入以下命令回车：
   ```cmd
   oobe\bypassnro
   ```
3. 电脑会自动重启，回到初始界面
4. 再次到联网步骤时，会出现**"我没有 Internet 连接"**选项，点击它
5. 选择**"继续执行受限设置"**
6. 接下来就能直接创建**本地账户**了

---

## 方法二：安装完成后，从微软账户切换为本地账户

**如果你已经用微软账户登录了：**

1. 打开 **设置 → 账户 → 你的信息**
2. 点击**"改用本地账户登录"**
3. 验证当前微软账户密码（或 PIN）
4. 设置本地账户用户名和密码
5. 注销后用本地账户登录
6. 可选：再到设置里删除原来的微软账户关联

---

## 方法三：新建一个本地账户（保留原账户）

1. **设置 → 账户 → 其他用户 → 添加账户**
2. 点击**"我没有这个人的登录信息"**
3. 点击**"添加一个没有 Microsoft 账户的用户"**
4. 输入用户名、密码、密保问题
5. 创建完成后，可将其设为管理员：
   - 点击该账户 → **"更改账户类型"** → 选**"管理员"**

---

## 快速对比

| 场景 | 方法 |
|---|---|
| 全新安装，不想绑微软账户 | **方法一**（oobe\bypassnro） |
| 已用微软账户，想换回本地 | **方法二** |
| 想保留微软账户，另开本地账户 | **方法三** |

---

**推荐**：如果你还没安装完系统，直接用方法一最省事，全程本地账户，干净无绑定。

## 推荐理由

| 版本 | 是否推荐 | 原因 |
|---|---|---|
| **Win11 专业版** ✅ | **首选** | 功能完整，Hyper-V/远程桌面/BitLocker 都有，激活成本合理 |
| Win11 家庭版 | 能用但不推荐 | 缺 Hyper-V、组策略、远程桌面主机，后期折腾受限 |
| Win11 专业工作站版 | ❌ 不推荐 | 你硬件远未触达其优势区间，多花冤枉钱 |
| Win10 专业版 | 备选 | 如果担心 Win11 兼容性，但新硬件建议直接 Win11 |

---

## 你这配置在 Win11 专业版下的表现

| 场景 | 预期表现 |
|---|---|
| 日常办公/编程 | 流畅无压力 |

Windows 11 专业工作站版（Pro for Workstations）与 Windows 11 专业版（Pro）的核心区别如下：

---

## 定位差异

| | **Win11 专业版** | **Win11 专业工作站版** |
|---|---|---|
| **目标用户** | 普通办公、中小企业、开发者 | 高端工作站、专业创作者、工程师、数据科学家 |
| **硬件场景** | 普通台式机/笔记本 | 服务器级工作站、多路CPU、大内存配置 |

---

## 工作站版独有的核心特性

| 特性 | 说明 |
|---|---|
| **ReFS 文件系统** | 弹性文件系统，支持数据自动修复，适合大型存储池和关键数据 |
| **持久内存支持** | 直接访问 NVDIMM-N 等持久内存设备，兼顾速度与数据持久性 |
| **SMB Direct** | 支持 RDMA 网卡的高速网络文件传输，降低 CPU 占用 |
| **多路处理器扩展** | 支持最多 **4 颗物理 CPU**（专业版仅支持 2 颗） |
| **内存上限提升** | 支持最高 **6TB RAM**（专业版上限 2TB） |
| **远程直接内存访问 (RDMA)** | 高性能计算网络支持，适合集群/服务器环境 |

---

## 两者共有的功能

- BitLocker 设备加密
- 远程桌面主机
- Hyper-V 虚拟化
- 组策略编辑器 (gpedit.msc)
- Azure AD 加入
- Windows Sandbox
- 分配访问权限/展台模式

---

## 选择建议

| 场景 | 推荐版本 |
|---|---|
| 日常办公、编程、轻度设计 | **专业版**（性价比更高） |
| 4K/8K 视频剪辑、3D 渲染、CAD/CAM、科学计算、大型数据库、服务器级硬件 | **专业工作站版** |

---

**一句话总结**：工作站版不是功能更多的系统，而是**对高端硬件支持上限更高的专业版**。如果你的电脑不是多路CPU + 大内存 + 专业存储的配置，专业版完全够用。

|---|---|---|
| **目标用户** | 普通办公、中小企业 | 高端工作站、专业创作者、工程师 |

## 核心功能区别

**专业工作站版独有的关键特性：**

- **ReFS 文件系统** — 弹性文件系统，数据恢复能力更强，适合大型数据存储
- **持久内存支持** — 直接访问 NVDIMM 等高速存储设备
- **SMB Direct** — 高速网络文件传输，支持 RDMA 网卡
- **多路处理器支持** — 支持最多 4 颗 CPU（专业版仅支持 2 颗）
- **更大内存上限** — 支持最高 6TB 内存（专业版最高 2TB）
- **远程直接内存访问 (RDMA)** — 高性能计算网络支持

## 共同点

两者都包含：
- BitLocker 加密
- 远程桌面主机
- Hyper-V 虚拟化
- 组策略管理
- Azure AD 加入
- Windows Sandbox

## 选择建议

- **普通用户/办公** → 专业版足够，性价比更高
- **视频剪辑/3D 渲染/科学计算/大型数据库** → 专业工作站版，硬件扩展性和数据可靠性更好

简单说：**专业工作站版是专业版的"硬件扩展增强版"**，软件功能基本一致，但对高端硬件的支持上限大幅提升。
