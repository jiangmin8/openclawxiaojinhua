不是的，这个设计**不是按"有问题/没问题"来分**的。

## 实际分类依据

| 分类维度 | 具体判断标准 |
|---------|-------------|
| **发件人身份** | 企业域名 vs 个人邮箱 |
| **邮件内容特征** | 含促销词汇、订单号、验证码模板 |
| **发送行为** | 批量发送标记 vs 单独发送 |
| **你的历史操作** | 常打开的进收件箱，常忽略的进推广 |

## 为什么你会感觉"有问题"？

- **收件箱** = 苹果认为"你可能想看的"
- **其他分类** = 苹果认为"批量发送的、不那么紧急的"

但算法会误判：
- 银行账单可能被丢进"推广"
- 朋友发来的团购链接可能被当"交易"
- 工作邮件如果来自群发系统，可能进"更新"

## 快速验证是不是误判

看邮件顶部的**灰色小标签**（iOS 16.4+）：
- 显示「推广」「交易」等字样 = 苹果自动分类的
- 没有任何标签 = 纯收件箱

---

**结论**：这不是安全机制，是**效率整理工具**。如果担心漏掉重要邮件，建议：
1. 关闭分类直接看传统收件箱
2. 或者养成习惯定期扫一眼「所有邮件」视图

你觉得哪些邮件被错分了？可以告诉我具体情况，我帮你判断是不是算法问题。

这是 iOS 邮件应用的智能分类功能，不是 bug。苹果在 iOS 16/17 之后引入了**邮件分类（Mail Categories）**，会自动把邮件分到不同标签页：

## 分类逻辑

| 标签页 | 收录规则 |
|--------|----------|
| **收件箱（Inbox）** | 个人邮件、熟人发来的、你经常互动的联系人 |
| **未读（Unread）** | 所有未读邮件的聚合视图，跨分类显示 |
| **推广（Promotions）** | 营销邮件、优惠券、广告 |
| **交易（Transactions）** | 订单确认、物流通知、账单收据 |
| **更新（Updates）** | 新闻订阅、App 更新通知、系统提醒 |

## 为什么同一发件人会有分歧？

- **发件人更换了邮件模板**（营销邮件 vs 客服回复）
- **邮件头部信息不同**（批量发送 vs 单独发送）
- **你之前的互动行为**（是否打开过类似邮件、是否回复过）

## 如果你想控制这个行为

**关闭自动分类：**
设置 → 邮件 → 邮件分类 → 关闭「按分类整理」

**手动纠正分类：**
在邮件上左滑 → 点击「移动到」→ 选择正确的分类，系统会学习你的偏好

**把重要联系人设为 VIP：**
邮件详情 → 点击发件人名字 → 添加到 VIP，这样他们的邮件永远优先显示在收件箱

---

简单说：**苹果在帮你过滤垃圾，但算法不完美**，重要邮件被错分了就手动移一次，系统会记住。
