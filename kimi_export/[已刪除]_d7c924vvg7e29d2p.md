## 高通版 AX3000 刷机（根据搜索结果 [^15^][^16^]）

**免拆机？不行。** 高通版必须拆机接 TTL 线刷分区文件。

### 刷机步骤：

1. **拆机焊接 TTL 排针**（或找免焊接触点）
2. **TTL 线连接**：TX/RX/GND 接好，**VCC 不接**
3. **波特率 115200**，上电快速按回车进 Uboot
4. **刷入分区文件**：
   ```bash
   tftpboot OpenWrt.mtd1.bin && flash 0:MIBIB
   tftpboot OpenWrt.mtd11.bin && flash 0:APPSBL
   tftpboot OpenWrt.mtd12.bin && flash 0:APPSBL_1
   ```
5. **刷 OpenWrt**：ubiformat 刷入 factory.ubi
