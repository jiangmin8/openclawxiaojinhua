## 🖥️ 网页端界面布局

### 左侧边栏 - 功能导航
| 功能 | 说明 |
|------|------|
| **网站** | 创建和编辑网页 |
| **文档** | 生成 Word/PDF 等文档 |
| **PPT** | 一键生成演示文稿 |
| **表格** | 智能生成 Excel 表格 |
| **深度研究** | 复杂调研任务 |
| **Kimi Code** | 编程辅助（限时活动） |
| **Kimi Claw** | 部署到第三方平台（Beta） |
| **历史会话** | 查看过往对话记录 |

### 中间区域 - 对话主界面
- **输入框**：输入问题或指令，支持上传图片
- **功能按钮**：
  - 📎 **+ 号**：上传文件/图片、设置常用语、联网搜索
  - 🤖 **Agent**：切换智能代理模式
  - **快捷入口**：网站、文档、PPT、表格、深度研究、Agent 集群

### 右上角 - 模型切换
可切换不同的 AI 模型：
- **K2.5 快速**：快速响应
- **K2.5 思考**：多轮搜索思考，回答复杂问题
- **K2.5 Agent**：调研、PPT、网站、文档、表格
- **K2.5 Agent 集群**（Beta）：海量搜索、长文写作、批量处理

### 底部 - 会员等级
左下角显示当前会员等级，点击可查看会员权益和升级选项。

## 📱 移动端界面

手机浏览器访问时，界面适配为移动端布局：
- **个人设置页**：管理个人资料、会员计划、账号安全、记忆空间、常用语等
- **历史会话**：查看过往对话
- **首页案例**：快速使用示例
- **语音输入**：按住说话进行语音输入
- **上传文件/打电话/常用语/联网设置**：快捷操作面板
