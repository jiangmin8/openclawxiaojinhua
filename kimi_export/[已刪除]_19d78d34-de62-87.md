### 2. **VLAN ID 设置**
部分地区光纤需要 **VLAN ID**，路由器拨号时要填写：

| 运营商 | 常见 VLAN ID |
|--------|-------------|
| 电信 | 41 |

### 3. **MAC 地址克隆**
运营商可能绑定了光猫的 MAC 地址，需要路由器 **克隆 MAC**：

```
路由器设置 → 网络参数 → MAC地址克隆 → 克隆光猫的MAC
```

---

### 4. **检查物理连接**
```
光纤 → 光猫 → 网线 → 路由器 WAN口
```
- 网线必须插在路由器 **WAN口**（通常蓝色）
