我来搜索一下 Hermes Agent 与微信连接部署的相关资料。

有的，Hermes Agent 有完整的**个人微信（Weixin）接入方案**，走的是**腾讯官方 iLink Bot API**，不是第三方破解协议。以下是部署要点：

---

## 一、前置条件

- 个人微信账号
- 已安装 Python 环境
- 安装依赖包：

```bash
pip install aiohttp cryptography
# 可选：终端显示二维码
pip install qrcode
```

---

## 二、接入步骤

### 1. 运行网关配置向导

```bash
hermes gateway setup
```

选择 **Weixin**，向导会自动：
- 向 iLink Bot API 请求二维码
- 在终端显示二维码（或提供链接）
- 等待你用手机微信扫码并确认登录
- 自动保存凭证到 `~/.hermes/weixin/accounts/`

登录成功显示：
```
微信连接成功，account_id=your-account-id
```

### 2. 配置环境变量

编辑 `~/.hermes/.env`，至少填上 account_id：

```bash
WEIXIN_ACCOUNT_ID=your-account-id

# 可选：限制私聊权限
WEIXIN_DM_POLICY=allowlist
WEIXIN_ALLOWED_USERS=user_id_1,user_id_2

# 可选：群聊控制（默认关闭）
WEIXIN_GROUP_POLICY=allowlist
WEIXIN_GROUP_ALLOWED_USERS=group_id_1

# 可选：定时任务/通知主频道
WEIXIN_HOME_CHANNEL=chat_id
WEIXIN_HOME_CHANNEL_NAME=Home
```

### 3. 启动网关服务

```bash
hermes gateway          # 前台运行
hermes gateway install  # 安装为后台常驻服务
hermes gateway start    # 启动服务
```

启动后，手机微信给 Bot 发消息，几秒钟内就能收到 AI 回复，对话框里还能看到"正在输入"状态。[^8^][^11^]

---

## 三、两种微信接入方式对比

| 方式 | 适用场景 | 稳定性 | 风险 |
|------|---------|--------|------|
| **个人微信 Weixin**（iLink Bot API） | 个人使用、轻量场景 | 官方接口，较稳定 | 建议先用小号测试 |
| **企业微信 WeCom** | 公司/团队办公自动化 | 更合规、更稳定 | 需企业微信管理员权限 |

企业微信有两种模式：
- **Bot 模式**：WebSocket 实时通信，无需公网回调地址
- **Callback 自建应用模式**：更正式，需公网服务器和回调配置 [^4^][^9^]

---

## 四、重要提醒

1. **建议先用微信小号测试**，确认稳定后再决定是否挂主号
2. 微信单条消息上限约 4000 token，长回复会自动分段
3. 密钥（API Key、Token）存到 `~/.hermes/.env`，普通配置存 `config.yaml`
4. Windows 原生环境不支持，必须在 WSL2 或 Linux 服务器上运行 [^3^][^4^]

这是 Hermes Agent 配置向导的下一步，问**群聊（Group Chats）怎么处理**。

三个选项：

1. **Disable group chats (recommended)** — 禁用群聊（推荐）。Hermes 不会参与任何群聊。
2. **Allow all group chats** — 允许所有群聊。Hermes 会响应所有群聊消息。
3. **Only allow listed group chat IDs** — 只允许白名单里的群聊 ID。

**建议选第一个（已高亮）**，按 **Enter** 确认。  
理由：Hermes 主要是你的个人 AI 助手，不是群聊机器人。开启群聊容易被滥用，也消耗 API 额度。如果以后有特定群聊需要它加入，再单独配置即可。

这是 Hermes Agent 的配置向导，在问**私信（Direct Messages）的授权方式**怎么设置。

四个选项：

1. **Use DM pairing approval (recommended)** — 使用配对审批（推荐）。有人要给你发私信时，需要先经过你同意/配对确认。
2. **Allow all direct messages** — 允许所有私信，任何人都能直接发给你。
3. **Only allow listed user IDs** — 只允许白名单里的用户 ID 发私信。
4. **Disable direct messages** — 完全关闭私信功能。

**建议选第一个（已高亮）**，按 **Enter** 确认即可。这是最安全的方式，避免被陌生人骚扰。
