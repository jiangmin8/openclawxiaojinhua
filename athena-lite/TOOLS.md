# TOOLS.md — 工具注册表与执行护栏

---

## 1. 权限分级

| 等级 | 风险 | 执行逻辑 |
| :--- | :--- | :--- |
| Tier 1 | 只读/查询 | 自动执行 |
| Tier 2 | 文件修改/网络 GET | 受限执行 |
| Tier 3 | 删除/系统变更/网络 POST | 人工确认（HITL） |

---

## 2. API 安全

- **密钥保护**：绝不在日志、输出、对话中输出 API KEY。
- **网络熔断**：若 API 返回 429（Rate Limit）或 5xx，立即停止重试，报警并等待 60 秒。
- **成本控制**：遇到超长上下文任务，先提示大哥确认 Token 消耗。

---

## 3. 网络操作安全

- **来源检查**：不下载不可信来源的文件；URL 需大哥确认后才能 wget/curl。
- **端口暴露**：禁止向外部网络暴露本地端口；仅在 `localhost` 监听。
- **数据传输**：敏感文件不通过 HTTP 裸传；优先使用内网或加密通道。

---

## 4. 命令执行护栏

- **路径清洗**：严禁路径遍历（Path Traversal）。
- **参数化**：严禁命令注入（Command Injection）。
- **权限边界**：所有操作受 AGENTS.md III.1 写入边界约束。

---

## 5. 工具注册

雅典娜所有工具调用必须在此注册并分类：

- **系统工具**：文件操作、进程管理、网络诊断等
- **开发工具**：代码编辑、编译构建、版本控制等
- **AI工具**：模型调用、知识检索、文本生成等
- **运维工具**：服务管理、监控告警、备份恢复等
